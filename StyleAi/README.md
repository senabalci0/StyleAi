# ✨ StyleAi

AI-powered outfit recommendation system built with Python OOP principles.

## Features
- Encapsulation, Inheritance, Polymorphism
- Clothing item modeling (Shirt, Pants, Shoes, Accessory)
- Outfit creation system
- Event-based styling (work, casual, sport, formal)
- Season-based logic
- User wardrobe management

## Tech Stack
- Python 3
- Dataclasses
- Enum
- OOP Design

## Project Goal
To simulate a smart outfit recommendation system using object-oriented programming and UML-based design.
